## Жобаның сипаттамасы
RAHKFC — бұл контейнеризацияланған қосымшаны автоматты түрде жинау, тексеру және орналастыру процесін көрсетуге арналған оқу жобасы. Жоба DevOps және CI/CD принциптерін практикалық түрде қолдануға бағытталған.

---

## Жобаның мақсаты
- CI/CD конвейерін құру және баптау  
- Docker контейнерлері арқылы қосымшаны біріздендіру  
- GitHub Actions көмегімен автоматтандырылған build процесін іске асыру  
- Қосымшаны болашақта Kubernetes арқылы масштабтауға дайындау  

---

## Қолданылған технологиялар
- **VCS:** GitHub  
- **CI/CD:** GitHub Actions  
- **Контейнеризация:** Docker  
- **Орындау ортасы:** Docker контейнерлері  
- **Оркестрация (жоспарланған):** Kubernetes (K8s)
